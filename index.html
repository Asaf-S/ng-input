<ng-input>

Text Input Effects For Angular Angular directives for enhancing text input interactions

check article at Codrops

Install

Install using bower

$ bower install ng-input

Install using npm

$ npm install ng-input

Usage

<ng-input theme='haruki' label='First Name' ng-model='first_name'></ng-input>
